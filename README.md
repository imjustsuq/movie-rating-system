# 🎬 Movie Rating System in Python

A simple Python application for registering and rating movies directly from the terminal.

## 📌 Features

- Add movies with a title and rating
- Display all registered movies
- Calculate the average rating
- Save movie data to a `.txt` file

## ⚙️ How It Works

The application uses:

- Loops to navigate through the program
- Conditional statements for flow control
- Functions to organize the code
- A `.txt` file to save review data

## 💡 Project Goal

This project was created to practice programming logic and fundamental Python concepts, including:

- Code organization
- Data manipulation
- User input and output
- Basic file handling

## 🚀 Possible Future Improvements

- Remove registered movies
- Edit existing movie information
- Create a more user-friendly terminal interface
- Organize movies by category or genre

## 🧑‍💻 Technologies Used

- Python

## 🔗 How to Run

1. Download the `.py` file.
2. Make sure Python is installed.
3. Run:

```bash
python project_movies.py
```
